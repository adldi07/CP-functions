# CP Functions Repository

This repository is for storing frequently used functions and templates for Competitive Programming (CP). Add your commonly used algorithms here for quick access during contests.

## Features
- Ready-to-use implementations of popular CP algorithms and utilities.
- Easy to extend with new functions as needed.

## Usage
1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Copy or include the required function files in your contest solutions.
3. Compile and run your code as usual.

## Contributing
Feel free to add new functions or improve existing ones. Keep code concise and well-documented for easy reuse.
